      How it works

      Content-Based

      Builds a TF-IDF vector from each movie's genres, overview, director, cast, and keywords. Ranks candidates by cosine similarity to the query movie.

      Collaborative (SVD)

      Decomposes the 671-user × 45,000-movie rating matrix into 50 latent factors using TruncatedSVD. Predicts unseen ratings and surfaces the highest-scoring unrated movies for a given user.

      Hybrid

      Weighted blend of normalised content similarity and normalised SVD collaborative score. Default weights: 60% content + 40% collaborative.

      📊 Classification Models

      Logistic Regression, Decision Tree, and Naive Bayes predict whether a user will "like" a movie (rating ≥ 4.0). Features: vote average, vote count, popularity, and genre one-hot encoding.

      About this project

      An end-to-end movie recommendation system built on The Movies Dataset (Kaggle, rounakbanik). The pipeline covers data cleaning, exploratory analysis, classification modelling, and three recommendation engines.

      Built by Subin Thapa as a BDS project at Tribhuvan University, Nepal.
